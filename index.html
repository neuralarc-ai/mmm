Media Mix Model Framework

Mathematical foundations and formulas for media mix modeling in India

Core MMM Equation

Salest = Base_Salest + Σβi × Saturation(Adstock(Mediai,t)) + Σγj × Controlj,t + εt

Where:

  • Salest = Total sales at time t
  • Base_Salest = Organic sales without marketing
  • βi = Media channel coefficient
  • Saturation() = Diminishing returns function
  • Adstock() = Carryover effects function
  • Controlj,t = External factors (seasonality, pricing)
